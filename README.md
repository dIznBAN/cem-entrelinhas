# EntreLinhas Podcast

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o projeto:
```bash
npm start
```

## Estrutura

- **Home**: Página inicial
- **Search**: Lista todos os episódios disponíveis
- **Episode Details**: Tela específica para cada episódio (quando clica em "Oficialato" ou "Projetos")
- **Player**: Reproduz o episódio selecionado

## Fluxo de navegação

1. **Navbar/Search** → **Episode Details** → **Player**
2. Cada episódio tem sua própria tela de detalhes
3. Botão "Reproduzir" leva para o player específico do episódio
